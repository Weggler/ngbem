We consider a perfect electric conducter \(\Omega \subset \mathbb R^3\) and a plane wave \(\boldsymbol E^i\) with trace \(\gamma_D \boldsymbol E^i = \boldsymbol m\) on its boundary \(\Gamma\). The trace \( \boldsymbol m\) induces a scattered electric field \(\boldsymbol E\) subject to the following Dirichlet boundary value problem:

\[\begin{split} \left\{ \begin{array}{rcl l} \mathbf{curl} \, \mathbf{curl}\, \boldsymbol E - \kappa^2 \, \boldsymbol E &=& \boldsymbol 0, \quad &\textnormal{in } \Omega^c \subset \mathbb R^3\,,\\ \gamma_D \,\boldsymbol E &=& \boldsymbol m, \quad & \textnormal{on }\Gamma \\ \left\| \mathbf{curl} \, \boldsymbol E( x) - i\,\omega\,\epsilon \, \boldsymbol E( x)\right\| &=& \mathcal O\left( \displaystyle \frac{1}{\| x\|^2}\right), &\textnormal{for} \; \|x\| \to \infty\,.\end{array} \right. \end{split}\]

The scattered field allows for a representation in terms of a single layer potential with unknown density \(\boldsymbol j\):

\[ \boldsymbol E(x) = \underbrace{\int_\Gamma \displaystyle{ \frac{1}{4\,\pi} \, \frac{e^{-i\,\kappa\,\|x-y\|}}{\| x-y\|} } \, \boldsymbol j(y)\, \mathrm{d}\sigma_y }_{\displaystyle{\boldsymbol S_\kappa (\boldsymbol j) } } + \frac{1}{\kappa^2} \nabla \underbrace{ \int_\Gamma \displaystyle{ \frac{1}{4\,\pi}\, \frac{e^{-i\,\kappa\,\|x-y\|}}{\| x-y\|} } \, \mathrm{div}_\Gamma \boldsymbol j(y)\, \mathrm{d}\sigma_y }_{\displaystyle{S_\kappa (\mathrm{div}_\Gamma \boldsymbol j) } } \,.\]

The density \(\boldsymbol j\) is determined by the boundary element method, i.e. the numerical solution of the variational formulation $\( \begin{array}{c} \forall \, \boldsymbol v\in H^{-\frac12}(\mathrm{div}_\Gamma, \Gamma): \quad \langle \boldsymbol S_\kappa (\boldsymbol j) + \frac{1}{\kappa^2} \nabla_\Gamma S_\kappa(\mathrm{div}_\Gamma \boldsymbol j), \boldsymbol v \rangle_{-\frac12} = \left\langle \boldsymbol m \times \boldsymbol n, \boldsymbol v\right\rangle_{-\frac12} \,. \end{array}\)$

In the enineering community, the approximation scheme is also known as method of moments (MoM) and the resulting equation for \(\boldsymbol j\) is called electric field integral equation (EFIE).

Define the geometry of the perfect conductor \(\Omega\) and create a mesh:

sp = Sphere( (0,0,0), 1)
mesh = Mesh( OCCGeometry(sp).GenerateMesh(maxh=1, perfstepsend=meshing.MeshingStep.MESHSURFACE)).Curve(2)

Next, we create two finite element spaces for \(\boldsymbol H^{-\frac12}(\mathrm{curl}_\Gamma, \Gamma)\) and rotate them later to obtain \(\boldsymbol H^{-\frac12}(\mathrm{div}_\Gamma, \Gamma)\) conforming shape functions as test and trial spaces.

fesHCurl = HCurl(mesh, order=2, complex=True)
uHCurl,vHCurl = fesHCurl.TnT() # H(curl_Gamma) conform spaces
print ("ndof HCurl = ", fesHCurl.ndof)

Define the incoming plane wave and compute the given boundary data \(\boldsymbol m\):

eps0 = 8.854e-12 
mu0 = 4*pi*1e-7
omega = 1.5e9
kappa = omega*sqrt(eps0*mu0)
print("kappa = ", kappa)

E = CF((1,0,0))*exp( -1j * kappa * z )
n = specialcf.normal(3)
m = Cross(n,E) # nxE in H(div_Gamma)

Compute the right hand side vector:

rhs = LinearForm( Cross(m,n) * Cross(n,vHCurl.Trace()) *ds(bonus_intorder=3)).Assemble() # <H(curl_Gamma), H(div_Gamma)>

The discretisation of the above variational formulation leads to a system of linear equations, ie

\[ \left( \mathbf S_\kappa - \frac{1}{\kappa^2} \,\mathrm S_\kappa \right) \mathbf m = \mathbf{rhs}\,,\]

where the single layer operator in the brackets on the left is a regular and symmetric matrix.

j = GridFunction(fesHCurl) # note: we actually solve for coefficients for FE space H(div_Gamma)
pre = BilinearForm(uHCurl.Trace()*vHCurl.Trace()*ds).Assemble().mat.Inverse(freedofs=fesHCurl.FreeDofs()) # to be changed
with TaskManager(): # pajetrace=1000*1000*1000):
    SVec = MaxwellVecSingleLayerPotentialOperator(fesHCurl, kappa, 
                                               intorder=12, leafsize=40, eta=3., eps=1e-4, method="svd", testhmatrix=False)
    SSca = MaxwellScaSingleLayerPotentialOperator(fesHCurl, kappa, 
                                               intorder=12, leafsize=40, eta=3., eps=1e-4, method="svd", testhmatrix=False)
    CG(mat = SVec.mat - 1/(kappa*kappa)*SSca.mat, pre=pre, rhs = rhs.vec, sol=j.vec, 
       tol=1e-8, maxsteps=500, initialize=False, printrates=True)
